Madat - Help comes to you!

The main goal of this website is to be able to have an as widespread reach as possible. We would like to be able to integrate and collaborate with many if not all the social organizations and NGOs which work for the welfare and betterment of this sector of society.
A lot of people, including students, farmers, pensioners atc., struggle with various day-to-day financial difficulties which end up hindering their dreams. The government as well as numerous Non Governmental Organizations, provide a lot of schemes for this very purpose, however they don't reach the intended audience due to one reason or another.
MADAT-Help Comes to You! aims to integrate all of these schemes under a single platform, thus making it easier and more accessible to the layman user. Here the user can view the various schemes and the categories they belong to. Further, the user can also check their eligibility for the scheme and apply for it.
